## how to write score
譜面の書き方。譜面内と譜面外で若干違う。

### 共通
|Prefix|内容|
|:---|:---|
|`#`|コメント|


### 譜面外
|Prefix|内容|例|
|:---|:---|:---|
|`:`|曲や譜面のプロパティを設定する。|`:bpm 120`|
|`!`|d|d|
|`[start]`|曲を開始する。|

### 譜面内(`[start]`よりあと)
|Prefix|内容|例|
|:---|:---|:---|
|`*`|ミリ秒と秒。|`*27.750`=`27.750sec`|
|`｜`(半角)|分|`｜3`|
|`@`|セクション。曲の区切り。|`@1-Verse`|
|`:`|歌詞|`スパイダーマッ！ﾃｰｯﾃﾃｯ！`|
|`:`|ふりがな|`:すまいだーまってーっててっ`|
|`[end]`|曲を終了する。|
